# reptrack

Gym assistant to track your exercise repitions, sets and be able to share them easily with your coach.

## Installation

1. Clone the Repository

2.

```
cd client
npm install
```

## Usage

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
